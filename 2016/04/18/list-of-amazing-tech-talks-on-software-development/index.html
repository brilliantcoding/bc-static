List of Amazing Tech Talks on Software Development

April 18, 2016April 18, 2016 Matthew Jackowski
Tweet

Building computer software is all about ideas. I find quite a bit of inspiration from listening to presentations and tech talks from other people in software development. You should be warned that some people might find the topics below fairly controversial!  But I feel that’s good. To me, if a presentation makes you pause to think and reflect, it is always worth the time to stop and listen.

Null References The-Billion Dollar Mistake via InfoQ


Is having the ‘null’ concept a useful abstraction in modern programming? Let’s hear from the person who invented it, Tony Hoare.

Additional information about Tony Hoare can be found here

 

 

Simple Made Easy via InfoQ


Rich Hickey brings a very different perspective to software design and architecture by suggesting that we keep our programs  ‘simple’.

Rich Hickey, is the benevolent dictator for life  for the Clojure programming language.

 

 

10+ Deploys Per Day: Dev and Ops Cooperation at Flickr via YouTube

Maybe development and operations should work together!? John Allspaw and Paul Hammond’s presentation are often cited as the beginning of the ‘DevOps’ movement.

The slides for this presentation can be found here via Slideshare

 

If there is a tech talk about software development and programming that should be on this list, please let me know in the comment section!
